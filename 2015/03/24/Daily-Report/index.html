3月24日的笔记

By Li

发表于 2015-03-24

文章目录

I go on reading the book which introduces level set method, and mainly focus on the mean curvature flow and some other methods for building the velocity field. Admittedly, it seems not accessible for me, since I cannot even understand the mathmatic expressions. But I still hold on, and endeavor to figure out the method, which is so useful on the image processing area.

Simultaneously, I also pay attention to the linux system, and in process of understanding the virtual memory allocation. Compared with the hardware structure introduced in the theorem of computer class, it is more abstract. To manage the RAM more effectively and enhance the efficiency, the physical memory is artificially divided into lots of logical segment called pages. During the executing of process, the kernel puts the corresponding code into specific pages, building a excutable to record the relation between the code and pages. When needed, the kernel can directly find corresponding pages and execute the code, leaving the need for putting all of codes into the RAM unnecessary. As a result, with the same memory space, there will be more processes executed.

Then, when it comes to the command line arguments, things become easier, since I have already dealt with similar matters. Still, the details mentioned in the book such as how the string of arguments end greatly increase my knowledge scope. And the sections that talk about the environmental variables help me gain a clear idea of a question that has confused me for a long time, which is how the environmental variables work.

Today, while I have spent almost the whole day in the lab, I don’t regard what I obtain enough. In other words, I should learn more. 时间并不算很多，是时候来点鸡血了！

Partial differential equations are truly difficult for me, as well as the convection equation. I should work harder on them.
